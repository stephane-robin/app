Héritage

Notion d'héritage

La notion d'héritage est issue de la programmation orientée objet. En CSS, cela signifie que si on applique une propriété à une balise, toutes les balises qui se trouvent à l'intérieur appliqueront cette même propriété. On dit qu'elles héritent de cette propriété. Par exemple si on définit la couleur du texte à l'intérieur d'un paragraphe comme étant bleue, tous les mots mis en valeur par la balise em à l'intérieur d'un paragraphe seront bleus, sauf si on redéfinit spécifiquement la couleur de la balise em.

Le code suivant est interprété de la façon suivante par le navigateur, où le mot "Attention" se trouvant à l'intérieur de balises <em> est colorié en bleu.

<p style="color:rgb(51,51,255)"><em>Attention</em>, je tape mon texte ici.</p>
HTML

associé à

p{
  font-family: Verdana, "Trebuchet MS", Arial;
  text-align: justify;
  color: rgb(51,51,255);
}
CSS

qui produit l'affichage suivant

Attention, je tape mon texte ici.

En revanche, le code suivant est interprété différemment, et la définition spécifique de la couleur de la balise <em>, outrepasse l'héritage issu de la balise <p>. 

<p style="color:rgb(51,51,255)"><em style="color:rgb(255,51,102)">Attention</em>, je tape mon texte ici.</p>

associé à

em{
  color: rgb(255,51,102);
}
CSS

qui produit l'affichage suivant

Attention, je tape mon texte ici.
