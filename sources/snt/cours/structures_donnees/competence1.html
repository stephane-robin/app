Utilité des bases de données

Tant qu'un programme est en cours d'exécution, il conserve en mémoire les valeurs des variables utilisées. En revanche, une fois qu'on stoppe ce programme ou qu'on se déconnecte d'un site Internet, les valeurs des variables sont perduées. Certaines ont pour vocation d'être temporaires, mais d'autres comme le montant du compte en banque, le niveau de jeu atteint, le mot de passe de connexion, ... doivent être conservées dans un serveur distant dont le rôle unique est de stocker des bases de données.

VOCABULAIRE ...

Une donnée est une valeur associée à un objet, un individu, un événement permettant de le décrire.

Le Big Data correspond à une quantité importante de données générées par l'utilisation de technologies numériques.

Une base de données est un ensemble de données reliées entre elles et structurées avec des tables.

Elements caractéristiques des bases de données

Les bases de données sont constituées de tables, elles-mêmes composées de lignes (ou enregistrements) et de colonnes (ou champs). Chaque table contient une clé primaire permettant d'identifier de manière unique chaque enregistrement. Par ailleurs, les différentes tables d'une base de données peuvent être reliées entre elles à l'aide de clés secondaires.

Voici un exemple de table d'une base de données, qui ressemble assez à un tableur Excel :

Ces données sont accessibles pour consultation et modification à l'aide d'un logiciel appelé SGDB Système de Gestion de Bases de Données.

Formats libres

Le format d'un fichier est un mode d'organisation des données qui les rend lisibles et permet de les mettre à jour en respectant un processus établi. Il est indiqué sous forme d'extension à la fin d'un fichier.

A vous de jouer : quels formats de fichiers connaissez-vous ?

Certaines données sont libres de droit, c'est à dire qu'il est possible de les télécharger et de les exploiter, mais pas de les modifier bien entendu. On parle alors de format ouvert pour les fichiers les contenant. Par exemple les fichiers aux extensions .xml, .csv, .txt, .odt ou .json sont des formats ouverts.

Un fichier enregistré dans un format ouvert est indépendant du logiciel utilisé pour le créer, ce qui le rend plus facile à lire, à modifier ou à imprimer pour un autre utilisateur ne disposant pas nécessairement du même logiciel que son auteur. Un format ouvert répond à un standard approuvé par une organisation internationale dans un but d'interopérabilité entre différents systèmes.

Le format .csv est très utilisé, notamment pour exporter des données d'un fichier libreOffice vers un fichier Excel. Il se caractérise par un délimiteur entre les champs comme une virgule ou un point-virgule, et chaque ligne représente un enregistrement.

Le format .json dérivé du langage JavaScript est également très utilisé du fait de la montée en puissance de ce langage Web.

A vous de jouer : entraînez-vous à importer un fichier du format .csv vers un tableur Excel ou libreOffice Calc.

Lire un fichier avec un programme Python

Le programme suivant permet de lire un fichier au format .csv directement dans l'environnement de développement, et de l'enregistrer dans maListe.

import csv
maListe = []

with open("monFichier.csv", 'r') as fichier:
	donnees = csv.reader(fichier, delimiter = ',')
	for ligne in donnees:
		maListe.append(ligne)
