Les variables

📚 Compétence : savoir définir puis modifier une variable

------------------------------

Qu'est-ce qu'une variable ?

Une variable permet de stocker une donnée du code. Elle est constituée d'un nom, d'une valeur et d'un emplacement en mémoire. En Python, on ne s'occupera que du nom de la variable et de sa valeur. A noter que le nom d'une variable ne peut pas commencer par un chiffre. Voici un premier exemple :

age = 18

On utilise le signe = pour affecter la valeur 18 à la variable appelée age. On peut ainsi changer la valeur de l'âge, qui est donc "variable".

Pour afficher la valeur d'une variable, il suffit d'utiliser l'instruction print avec le nom de cette variable :

print(age)

affichera 18.

Attention, vous avez déjà exécuté l'instruction suivante pour tester Python :

print("Hello world")

Notez l'importance des "" dans le script ci-dessus. Essayez le même script sans les "". L'interpréteur Python pense dans ce cas que Hello est une variable dont il ne connaît pas la valeur. Les "" permettent simplement d'afficher du texte et non pas la valeur d'une variable.

Les types de variables

Chaque variable appartient à une famille particulière relative à la valeur de cette variable :

  • les nombres entiers sont stockés dans des variables de type int
  • les nombres réels sont stockés dans des variables de type float
  • les textes (chaînes de caractères) sont stockés dans des variables de type str
  • les valeurs True et False sont stockées dans des variables de type bool

A noter que la place qu'occupe une variable en mémoire dépend fortement du type de cette variable.

On peut afficher le type d'une variable avec l'instruction type et le nom de la variable. Par exemple : 

print(type(age))

affichera int.

Exercice type : dans un jeu vidéo, le niveau peut varier de beginner à master. Affecter la valeur beginner à la variable Utilisateur, puis la valeur master à cette variable. Afficher ensuite le type de la variable Utilisateur.

Conventions de notation

Les noms de variables doivent avoir un sens et s'écrivent très souvent d'une des manières suivantes :

nomUtilisateur = "Antoine"
nom_utilisateur = "May"
PI = 3.14

On remarque que les constantes (qui ne varient donc pas) s'écrivent en majuscules.

Les opérateurs

Les opérateurs servent à modifier la valeur des variables. Pour les nombres entiers (int) et les nombres réels (float), on peut utiliser les opérations suivantes :

+ pour l'addition

x = 1
x = x + 1

affectera la valeur 2 à la variable x.


- pour la soustraction

a = 5
b = 2
x = a - b
affectera la valeur 3 à la variable x.

* pour la multiplication

x = 5 * 7

affectera la valeur 35 à la variable x.


/ pour la division

x = 7 / 5

affectera la valeur 1.4 à la variable x.


** pour l'exposant

x = 7 ** 2

affectera la valeur 49 à la variable x.


// pour le quotient d'une division

x = 7 // 5

affectera la valeur 1 à la variable x.


% pour le reste de division

x = 7 % 5

affectera la valeur 2 à la variable x. Attention la division décimale de 7 par 5 donne 1.4, mais la division euclidienne de 7 par 5 s'écrit 7 vaut 5 fois 1 plus 2. Ce qui explique le résultat obtenu. Les opérations sont similaires mais la division euclidienne permet de mettre en valeur l'opérateur modulo % qui va parfois s'avérer très utile.
