La boucle while

📚 Compétence : savoir utiliser une boucle while

------------------------------

On utilise la boucle while lorsqu'on ne connaît pas le nombre de répétitions à effectuer et que celui-ci dépend de l'évolution de la situation (par exemple si le nombre de vies d'un joueur devient nul).

Par exemple, on peut répéter Continuez de jouer tant que la variable nbreVies est positive

nbreVies = 10
while nbreVies > 0:
    print("Continuez de jouer")
    nbreVies = nbreVies - 1

Le script ci-dessus correspond en fait au code Scratch suivant que vous connaissez bien

Structure d'une boucle while

Une boucle while se construit donc avec :

  • une condition qui renvoie True ou False,
  • une indentation pour indiquer le bloc de code à répéter,
  • une variable dont la valeur change pour modifier la condition au tour suivant.

Dans l'exemple ci-dessus, la condition nbreVies > 0 renvoie d'abord True puisque nbreVies vaut 10, puis cette condition est testée à chaque tour jusqu'à ce que nbreVies soit égal à 0. Lorsque la condition renvoie false, le programme n'effectue pas de tour supplémentaire et exécute la suite du code (après la boucle while donc après l'indentation). C'est la variable nbreVies qui varie pour modifier la condition à tester.

Le risque avec une boucle while est que la condition d'arrêt ne soit jamais remplie, ce qui ne permet pas à la boucle de se terminer. On crée ainsi une boucle infinie qui bloque le programme. Par exemple 

compteur = 0
while compteur >= 0:
    print("le compteur est positif")

Exercice type : afficher quel est votre nom ? tant que l'utilisateur ne saisit pas un nom au clavier.

Solution : 

nom = ""
while len(nom) == 0 
    nom = input("Quel est votre nom ?")

Python ne connaît pas de méthode isEmpty() comme d'autres langages. Il faut donc tester la longueur de la chaîne de caractères nom à la place.
