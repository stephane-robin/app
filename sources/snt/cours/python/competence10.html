TP - mot de passe

📚 Compétence :

------------------------------

Vous pourrez travailler par groupe de deux pour réaliser ce Travail Pratique. Il devra être rendu sur Teams dans la rubrique Devoirs en fin de séance.

Vous êtes développeur chez JMLinfo. Votre client vous demande de faire un script Python qui vérifie les mots de passe entrés pas l’utilisateur sur un site. Vous devrez :

  • demander à l’utilisateur de saisir un mot de passe de plus de 8 lettres contenant au moins une majuscule et un chiffre,
  • vérifier que le mot de passe saisi ne contient pas d’espace vide,
  • vérifier que le mot de passe saisi ne contient pas les caractères interdits <, >, /
  • vérifier que le mot de passe saisi contient plus que 8 caractères,
  • vérifier que le mot de passe saisi contient au moins une majuscule,
  • vérifier que le mot de passe saisi contient au moins un chiffre,
  • vérifier que le mot de passe est différent des mots de passe déjà utilisés "toTo888", "taTa888" et "tiTi888".

Ces vérifications doivent se faire en utilisant une condition if qui, si elle n’est pas remplie, ne permet pas d'accéder au site. Dans ce cas, le programme affiche "mot de passe incorrect". En revanche, si la condition est remplie, le programme affiche "bienvenue sur le site", Bonus : utiliser une boucle while à la place d’une condition if et, dans ce cas, répéter la demande de mot de passe tant que celui-ci ne respecte pas les critères demandés.
