TP - résolution de système

📚 Compétence :

------------------------------

Vous poucez travailler par groupe de deux pour réaliser ce Travail Pratique. Il devra être rendu sur Teams dans la rubrique Devoirs en fin de séance.

On cherche à résoudre une équation du type \(ax + b = cx + d\) où \(a, b, c, d \in \mathbb{N}\). Proposez un script Python permettant de résoudre cette équation. Il existe plusieurs approches de résolution (algébrique, graphique, ...), et toutes les solutions mathématiques justes sont acceptées. Vous devrez :

  • demander à l’utilisateur de saisir les valeurs de \(a, b, c, d\) au clavier (indic. : pensez à input),
  • vérifier que les valeurs de \(a, b, c, d\) saisies soient bien des nombres entiers,(indic. : pensez à isnumeric()),
  • analyser les solutions possibles en fonction des valeurs données (indic. : pensez à la condition if)
  • afficher le résultat pour l’utilisateur (indic. : pensez à print),
