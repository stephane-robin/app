Les chaînes

📚 Compétence : savoir manipuler les chaînes

------------------------------

Un texte est composé de caractères. En informatique, on utilise le terme chaîne de caractères à la place de texte et on numérote chaque caractères en partant de l'indice 0. Python associe à une chaîne de caractères le type str pour string.

A RETENIR ...

Dans la chaîne de caractères bonjour, la lettre b a pour indice 0 et non pas 1, c'est à dire qu'elle se trouve à la position 0.

La concaténation

Dans le cas d'une chaîne de caractères, l'opérateur + ne sert pas à additionner, mais à concaténer, c'est à dire coller un texte à la suite d'un autre. Par exemple 

print("Les oranges sont " + "des fruits")

affichera Les oranges sont des fruits sans procéder à aucun calcul.

Transformer des chaînes

méthodesignification
len(maChaine)renvoie le nombre de caractères de maChaine
maChaine.find("a")renvoie la 1ère position de la lettre a dans maChaine, renvoie -1 si maChaine ne contient pas la lettre a
maChaine.upper()renvoie une chaîne correspondant à maChaine en majuscules, attention la variable maChaine n'est pas modifiée
maChaine.capitalize()renvoie une chaîne correspondant à maChaine avec la 1ère lettre en majuscule, attention la variable maChaine n'est pas modifiée
maChaine.count("a")renvoie le nombre de fois que la lettre a apparaît dans maChaine
maChaine.replace("a", "b")renvoie une chaîne correspondant à maChaine dans laquelle les a ont été remplacés par des b, attention la variable maChaine n'est pas modifiée
maChaine.isnumeric()renvoie True si maChaine peut être transformée en nombre entier, False sinon

Extraire une chaîne

Il est possible d'extraire un string à partir d'un string. Exécutez le code suivant pour comprendre ce qui se produit 

maChaine = "la neige fond en été"
print(maChaine)
print(maChaine[2:6])
print(maChaine[2:])
print(maChaine[:4])
print(maChaine[-1])
print(maChaine[-5])
