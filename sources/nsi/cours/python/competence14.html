TP - convertisseur binaire

📚 Compétence :

------------------------------

L'objectif de ce travail pratique est d'écrire 4 fonctions Python permettant de mettre à jour nos connaissances.

  • une fonction decimalTObinaire() qui transforme l'écriture décimale d'un nombre en écriture binaire,
  • une fonction binaireTOsexagesimal() qui transforme l'écriture binaire d'un nombre en écriture sexagésimale,
  • une fonction sexagesimalTObinaire() qui transforme l'écriture sexagésimale d'un nombre en écriture binaire.
  • une fonction binaireTOdecimal() qui transforme l'écriture binaire d'un nombre en écriture décimale,

Une fois écrites, ces fonctions seront conservées pour être utilisées dans le cours sur les réseaux.
