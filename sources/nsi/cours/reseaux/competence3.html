Le Web

📚 Compétence : comprendre la différence entre le Web et Internet

------------------------------

Internet est un réseau informatique mondial distribué, composé de millions de sous-réseaux publics ou privés. De ce fait, il n'existe pas de terminal qui centralise les informations. Internet utilise des applications de messagerie instantanée, de courrier électronique, de partage peer-to-peer, ainsi que le Web.

Le World Wide Web, symbolisé par www, est un service particulier d'Internet. Il s'agit d'un système qui permet de naviguer de pages en pages en cliquant sur des liens. On utilise un navigateur ou browser (Chrome, Firefox, Edge, Opera, Safari,...) pour y accéder et pouvoir lire les différentes pages. Le navigateur est une application qui fait office de traducteur de code HTML, CSS et JavaScript sous forme de page lisible par l'utilisateur.

En résumé, Internet est un réseau, une infrastructure, alors que le Web est un service sur ce réseau.

Clients-serveurs

Les clients sont les ordinateurs qui servent à consulter les sites Web.

Les serveurs sont des ordinateurs qui offrent des services. Ils peuvent par exemple stocker des sites Web et des bases de données. Le fonctionnement du Web est basé sur une structure client-serveur.

Une requête est une demande émise par un ordinateur client à destination d’un serveur qui contient l’information recherchée.

Voyons ensemble comment s'effectue une requête lorsque vous consultez un site.

Tout d'abord, vous écrivez dans votre navigateur l'adresse URL du site que vous souhaitez consulter, par exemple :

Le protocole HTTP HyperText Transfert Protocol permet l'accès au Web et la communication entre un client et un serveur, en récupérant des données telles que les contenus HTML ou CSS. Sa version sécurisée est HTTPS.

Toutefois, le Web n'est pas capable d'interpréter cette adresse URL qui est conçue pour faciliter la navigation de l'utilisateur. Nous savons déjà qu'Internet utilise des adresses IP. C'est le rôle du serveur DNS de traduire un nom de domaine en adresse IP. Le serveur DNS agit donc comme un annuaire téléphonique en fournissant la correspondance entre un nom de domaine et son adresse IP.

Vous accédez maintenant au Web et le protocole TCP/IP permet les échanges des données sur Internet.
