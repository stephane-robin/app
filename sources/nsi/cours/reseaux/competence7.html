Ce travail pratique consiste à écrire le code Python permettant de chiffrer et déchiffrer des messages en utilisant la technique de chiffrement asymétrique RSA. Il n'est pas nécessaire de connaître les concepts mathématiques sous-jacents. Les formules utilisées seront fournies dans le sujet ainsi que les clés publiques et privées.
