TP - Python et SQLite

📚 Compétence :

------------------------------

L'objectif de cette séance est de mettre en pratique les connaissances vues dans le chapitre sur le langage SQL en utilisant le Système de Gestion de Bases de Données SQLite et le langage Python.

Vous pourrez trouver de l'aide sur le site SQLite tutorial.

Vous allez créer un programme Python grâce auquel :

  • vous devrez établir la connexion entre votre programme Python et le SGDB SQLite
  • créer une base de données appelée skis
  • créer les tables fournisseurs et competitions utilisées dans le chapitre Langage SQL
  • tester tous les codes SQL proposés dans le chapitre Langage SQL pour constater le bon fonctionnement du programme, qui devra à partir du langage Python passer des commandes SQL à la base de données skis
