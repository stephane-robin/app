Structures de données hiérarchisées

📚 Compétence : savoir déterminer les métriques relatives à un arbre

------------------------------

Un arbre est une structure de données hiérarchisée constituée de noeuds, qui possèdent le plus souvent un parent et un ou plusieurs enfants. On représente généralement un arbre à partir d'un sommet qui se disperse vers le bas. Pour comprendre les terminologies utilisées pour décrire un arbre, il faut donc l'imaginer renversé, avec la racine en haut et les feuilles en bas.

A RETENIR ...

Le sommet d'un l'arbre est appelé racine.
Une feuille est un noeud sans enfant. C'est donc un des bouts de l'arbre.
Une branche est une suite de noeuds consécutifs partant de la racine et se terminant en une feuille.
La taille d'un arbre est le nombre de noeuds qui le composent.
La hauteur ou profondeur d'un arbre correspond à la distance de la racine à la feuille la plus éloignée.

On remarque d'après ces définitions qu'un arbre en informatique a autant de branches que de feuilles.

Arbres binaires

Les arbres binaires sont constitués de noeuds pouvant avoir 0, 1 ou 2 enfants. Attention, pas 0 ou 1 mais dans ce cas 0, 1 ou 2. Dans ce type d'arbre, on distingue généralement le sous-arbre gauche noté SAG du sous-arbre droit noté SAD.

Complexité des algorithmes

Nous pourrons déterminer la complexité d'un algorithme associé à un arbre à partir des métriques suivantes :

  • la taille de l'arbre
  • la hauteur d'un noeud dans l'arbre
  • la hauteur de l'arbre
  • la longueur de cheminement de l'arbre, correspondant à la somme des hauteurs de tous les noeuds
  • la profondeur moyenne de l'arbre, correspondant au rapport de la longueur de cheminement par le nombre de noeuds de l'arbre

On peut résumer ainsi les formules correspondant aux principales métriques d'un arbre :

hauteur du noeud \(x_i\) : \(H(x_i) = 1 + H(x_{i-1})\)

hauteur de l'arbre :
\(H(arbre) = max(H(x_1), ..., H(x_n))\) pour tous les noeuds \(x_1, ..., x_n\)

longueur de cheminement de l'arbre :
\(LC = H(x_1) + ... + H(x_n)\) pour tous les noeuds \(x_1, ..., x_n\)

profondeur moyenne de l'arbre : \(PM = \frac{LC}{n}\) où \(n\) est la taille de l'arbre.

Exercice type : A partir de l'arbre ci-dessous déterminer les principales métriques

  • L'arbre possède 12 noeuds. Sa taille est donc \(T = 12\)
  • Les différentes feuilles sont D, F, G, K, M. L'arbre possède donc 5 branches
  • Hauteurs de certains noeuds : \(H(A) = 0, H(B) = 1, H(D) = 3\)
  • Hauteur de l'arbre : \(H(arbre) = 4 = H(M)\)
  • Longueur de cheminement de l'arbre : \(LC = H(A)+H(B) + H(C) + ... + H(M)\) \(LC = 0 + 1 + 2 + ... + 4 = 0 + 3 \times 1 + 3 \times 2 + 4 \times 3 + 1 \times 4 = 25 \)
  • Profondeur moyenne de l'arbre : \(PM = \frac{LC}{T} = \frac{25}{12} = 2,1\)
