Structure du code

CSS signifie Cascading Style Sheets. Ce langage gère l'apparence des pages Web en arrangeant le contenu (couleur, marges, taille du texte). Comme il travaille le contenu d'une page Web, CSS utilise le code HTML pour fonctionner. Sans aucun code CSS, la page Web ressemble à du texte regroupé par bloc et tapé "au kilomètre".

Structure d'une page CSS

Il est possible de coder la mise en forme d'une page Web directement dans un fichier HTML sans recourir à un fichier CSS. Dans ce cas, on définit individuellement pour chaque balise la forme adéquate à l'aide de l'attribut style.

<p style="color:red"> mon texte </p>
HTML

Cette définition locale est coûteuse en temps et n'est conseillée qu'à titre exceptionnel. On peut également définir la mise en forme dans l'en-tête head d'un fichier HTML. Toutefois, la méthode préconisée est de définir la mise en forme dans un fichier séparé comportant l'extension .css. Il faut alors rajouter l'instruction suivante afin de lier notre page HTML à notre page CSS.

<link rel="stylesheet" href="style.css">
HTML

On constate que l'attribut href pointe vers l'adresse de la page CSS qui devra se situer dans le répertoire courant.

Les commentaires en CSS sont entourés des symboles /* et */. Ils peuvent être écrits sur une ou plusieurs lignes.
