Un langage interprété

Python est un langage de programmation interprété dont la philosophie repose sur la simplicité du code et l'implémentation de bonnes pratiques. Rien n'est fait pour empêcher le programmeur de mener une action, même si celle-ci semble inappropriée. Dans un langage interprété, le code source est traduit pas à pas par un logiciel appelé interpréteur, qui récupère des données d'entrée pour fournir des données de sortie. Le code source n'est pas transformé en code binaire comme pour un langage compilé.

Le principal avantage est que le code source d'un langage interprété peut fonctionner sur tout ordinateur. Le principal inconvénient est qu'un langage interprété est plus lent qu'un langage compilé, qui peut réutiliser certaines parties de code préalablement compilées.

Grâce à ses bibliothèques performantes et régulièrement mises à jour, Python est particulièrement utilisé dans les environnements scientifiques, pour la manipulation de données et pour l'intelligence artificielle.

Environnement de développement

On peut exécuter un programme Python en utilisant le Terminal d'un ordinateur (sous Linux et MacOS), mais il est plus pratique d'utiliser un éditeur de texte tel que Sublim Text ou NotePad++ qui colorient automatiquement le code en fonction du langage choisi et proposent une auto-complétion des instructions. Il est encore plus efficace d'utiliser un Environnement de Développement Intégré IDE tel que PyCharm ou JetBeans qui proposent en plus une recherche passive des erreurs de frappe, un debugger (aide pour corriger les bugs) et le paramétrage des outils de versioning (pour conserver l'historique du code).

Le debugger est un outil particulièrement intéressant, qui propose non seulement des commentaires relatifs aux erreurs de code, des solutions possibles, mais également la possibilité d'exécuter le code de point d'arrêt en point d'arrêt pour connaître l'état des variables du programme à chacune des lignes de son exécution, ces lignes ayant été préalablement définies par le développeur.

Un outil de versioning tel que Git est également très utile puisqu'il permet de conserver un historique des différentes versions de code, de revenir en arrière en choisissant une version plutôt qu'une autre si le code actuel n'est pas satisfaisant. Par ailleurs, il permet de travailler sur des copies de code plutôt que sur des originaux et de regrouper le travail de toute une équipe en évitant les conflits de code.

Python est souvent installé nativement sur votre ordinateur, mais sa version permet uniquement de faire fonctionner certaines applications de votre machine. Elle est indépendante de l'éditeur de texte ou de l'Environnement de Développement Intégré que vous utilisez pour exécuter du Python. De ce fait, l'utilisation du langage Python pour exécuter du code nécessite en général que l'utilisateur paramètre sur son ordinateur le chemin d'accès à l'interpréteur Python.

Lorsqu'on débute en Python, ce paramétrage peut s'avérer laborieux et décourageant. C'est pour cette raison que nous conseillons, pour débuter en Python, de vous consacrer uniquement à la rédaction du code au lieu de passer du temps à installer et à configuration un Environnement de Développement fonctionnel. Nous vous proposons ainsi d'utiliser un interpréteur Python en ligne.

Les interpréteurs en ligne sont très pratiques, mais ont principalement deux inconvénients. Ils peuvent être lents suivant le traffic occasionné sur le site au moment de l'utilisation, et ils n'acceptent pas d'affichage en sortie avec une fenêtre indépendante du navigateur utilisé. Il devient dans ce cas compliqué de faire des dessins ou de créer des fenêtres pour une application.

Nous vous conseillons donc de démarrer avec un interpréteur Python en ligne, puis de rapidement passer à un éditeur de texte ou un environnement de développement. Pour cela, vous aurez besoin d'installer une version récente de Python sur votre ordinateur, puis vous devrez installer et configurer un outil tel que PyCharm, Sublim Text, Netbeans, Visual Studio Code ou Atom. En revanche, si vous utilisez déjà un IDE, c'est parfait, ne changez rien.

Exécuter son premier programme

L'exécution d'un programme se réalise avec la touche Run ou Exécuter sur les interpréteurs en ligne que vous venez de découvrir.

Il est esentiel de vérifier le bon fonctionnement d'un code simple avant de commencer le prochain chapitre.

A vous de jouer : par exemple, pour afficher "Hello world", exécutez le code suivant dans votre interpréteur en ligne :

print("Hello world")

Petit rappel : la touche Run ou Exécuter

Si tout fonctionne normalement, vous devez voir s'afficher à l'écran 

Hello world

Lorsque vous exécutez votre programme directement sur votre machine au lieu d'utiliser un interpréteur en ligne, il peut être utile de rajouter à la première ligne de votre code un shebang qui indique à l'ordinateur où se trouve dans votre machine la version du langage Python que vous souhaitez utiliser, comme dans l'exemple ci-dessous où Python se trouve dans le répertoire bin :

#!/usr/bin/python3

Pour enregistrer votre code, il faut penser à lui donner l'extention .PY, comme par exemple monFichier.py
