Fonctions Python

📚 Compétence : savoir définir et utiliser une fonction en Python

------------------------------

Définir une fonction

Pour définir une fonction, rien de plus simple, il faut :

  • utiliser le mot-clé def,
  • choisir les paramètres de la fonction, qui vont faire varier le résultat,
  • penser à indenter les instructions afin de créer un bloc, qui définit pour Python le début et la fin de la fonction
  • si une valeur est renvoyée à la fin, ne pas oublier le mot clé return.

Par exemple la fonction \(f\) telle que \(f(x)=2x+1\) se définit de la façon suivante en Python :

def f(x):
    return 2*x+1

Appeler une fonction

Une fois que la fonction \(f\) a été définie, pour obtenir par exemple la valeur de \(f(3)\), il suffit d'écrire

f(3)

Attention, dans ce cas l'interpréteur calcule la valeur de \(f(3)\) mais n'en fait rien. Si on veut que \(f(3)\) s'affiche à l'écran, on doit écrire

print(f(3))

et si on veut que \(f(3)\) soit enregistré comme valeur d'une variable maVariable, on doit écrire

maVariable = f(3)
