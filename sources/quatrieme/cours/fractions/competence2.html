TP - Mes débuts en Python

📚 Compétence : découvrir le langage Python

------------------------------

Python est un langage de programmation permettant des calculs scientifiques poussés et des représentations graphiques élaborées, grâce à l'utilisation de bibliothèques conçues à cet effet.

Faites parler l'ordinateur

Tout d'abord, il nous faut un interpréteur Python pour traduire le code en langage que l'ordinateur pourra comprendre. Pour faire simple, nous allons utiliser un interpréteur en ligne sur le site Python online gdb compiler.

Chaque instruction Python pourra maintenant être exécutée en appuyant sur le bouton Run

Par exemple, pour afficher Bonjour, il suffit d'exécuter le code suivant 

print("Bonjour")

Notez l'importance des guillemets, sinon cela entrâine un erreur. Envie de planter l'ordinateur. Enlevez un guillemet et exécutez le code. La console communique avec vous en vous diant ce qui ne va pas.

Bon maintenant, nous aimerions que l'ordinateur nous dise bonjour personnellement. Il faut donc rentrer votre nom dans une variable, puis l'afficher. C'est parti...

nom = input("Quel est votre nom ? ")
print("Bonjour", nom)

Voilà pour les bases de la communication en Python. En mathématiques, on utilise essentiellement des opérations.

Les opérateurs

Pour affecter la valeur 18,2 à la variable x, on écrit

x = 18.2

On utilise le signe = pour affecter la valeur 18,2 à la variable appelée x. On remarque également l'utilisation du point à la place de la virgule suivant la norme anglaise.

Pour afficher la valeur d'une variable, il suffit d'utiliser l'instruction print avec le nom de cette variable :

print(x)

affichera 18.2.

Les opérateurs servent à modifier la valeur des variables. Pour les nombres entiers (int) et les nombres réels (float), on peut utiliser les opérations suivantes :

+ pour l'addition

x = 1
x = x + 1

affectera la valeur 2 à la variable x.


- pour la soustraction

a = 5
b = 2
x = a - b

affectera la valeur 3 à la variable x.


* pour la multiplication

x = 5 * 7

affectera la valeur 35 à la variable x.


/ pour la division

x = 7 / 5

affectera la valeur 1.4 à la variable x.


** pour l'exposant

x = 7 ** 2

affectera la valeur 49 à la variable x.


// pour le quotient d'une division

x = 7 // 5

affectera la valeur 1 à la variable x.


% pour le reste de division

x = 7 % 5

affectera la valeur 2 à la variable x. Attention la division décimale de 7 par 5 donne 1.4, mais la division euclidienne de 7 par 5 s'écrit 7 vaut 5 fois 1 plus 2. Ce qui explique le résultat obtenu. Les opérations sont similaires mais la division euclidienne permet de mettre en valeur l'opérateur modulo % qui va parfois s'avérer très utile.

A vous de jouer : testez tous les code ci-dessus. C'est encore le meilleur moyen de comprendre ce qu'il se passe. Python est avant tout une formidable calculatrice.
