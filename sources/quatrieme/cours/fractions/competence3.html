TP - Calcul de fractions avec Python

📚 Compétence : apprendre à utiliser Python pour vérifier ses calculs

------------------------------

Vous avez appris à effectuer des opérations avec les fractions. C'est parfois difficile, mais c'est utile de savoir le faire afin de bien comprendre les différentes règles de calcul. Plus tard, vous pourrez alors appliquer ces principes au calcul littéral utilisant des fractions. En attendant, le langage Python permet également de trouver le résultat d'opérations simples sur les fractions, et cela à moindre effort. Une fois que vous saurez utiliser Python correctement, cet outil performant vous permettra par exemple de vérifier le résultat de vos calculs.

Pour bien démarrer

Nous allons utiliser un interpréteur Python en ligne pour exécuter le code. Pour commencer, il faut donc se rendre sur le site Python online gdb compiler.

Partie 1 - calcul manuel

Dans un premier temps, soyez courageux et effectuez les calculs suivants à la main, afin de comparer plus tard vos résultats et ceux de l'ordinateur.

\(\frac{2}{3}+\frac{7}{9} = ...\)

\(\frac{5}{2}-\frac{4}{7} = ...\)

\(\frac{5}{2}\times\frac{4}{7} = ...\)

\(\frac{7}{2}/\frac{3}{7} = ...\)

Partie 2 - Python

Pour exécuter les codes ci-dessous, vous pouvez appuyer sur le bouton Run de l'interpréteur Python.

Nous allons maintenant retrouver les résultats précédents en utilisant des codes simples du langage Python.

En Python le calcul de fractions nécessite l'importation d'un module appelé fractions. Pour cela, commencez votre code par

from fractions import Fraction

Pour calculer la somme \(\frac{1}{3}+\frac{1}{4}\) en Python il faut exécuter le code suivant :

print(Fraction(1, 3)+Fraction(1, 4))

Essayez maintenant d'obtenir à l'aide de Python les résultats des calculs que vous venez de faire manuellement. Pour rappel, la multiplication utilise le symbole * et la division le symbole / en Python.
