Saisie clavier

📚 Compétence : Savoir récupérer une information saisie par l'utilisateur

------------------------------

Afin de communiquer avec l'utilisateur du programme, il est nécessaire de pouvoir récupérer ce qu'il tape au clavier. Pour cela, on utilise l'instruction input. La question posée à l'utilisateur, et qui appelle une réponse de sa part doit être posée entre parenthèses suivant l'instruction input. Il est inutile dans ce cas d'utiliser l'affichage de la question avec print. Ainsi, le code suivant affiche la question Comment vous appelez-vous ? et attend une réponse. L'interpréteur Python est donc en pause jusqu'à ce que l'utilisateur saisisse au clavier sa réponse puis valide avec la touche Entrée.

input("Comment vous appelez-vous ?")

Attention, cette réponse est perdue si elle n'est pas affectée à une variable. Ainsi, le code ci-dessus doit être modifié de la façon suivante pour être utile :

nom = input("Comment vous appelez-vous ?")

Maintenant seulement, la valeur saisie au clavier par l'utilisateur sera réutilisable en appelant la variable nom

: 
print(nom)

A vous de jouer : demandez à l'utilisateur son âge puis affichez la valeur saisie en utilisant une variable age.

Le type par défaut saisi au clavier

Tout ce que saisit l'utilisateur au clavier est enregistré à l'aide de l'instruction input en tant que str, même un nombre. Essayez pour vous en convaincre le script suivant

age = input("quel est ton âge ? ")
print(type(age))

Afin de pouvoir garder en mémoire un nombre saisi par l'utilisateur, et donc de pouvoir lui faire subir des opérations, il faut obligatoirement le transformer en nombre. Pour cela, on utilise l'instruction suivante 

age = int(input("quel est ton âge ? "))
print(type(age))

On peut également utiliser le type float au lieu du type int si l'on souhaite transformer une châine de caractères en nombre réel, plutôt qu'en nombre entier. Attention toutefois, on ne peut transformer une chaîne de caractères en nombres que si elle est effectivement composée de nombres. Ainsi, l'instruction int("bonjour") entraînera une erreur.

A RETENIR ...

"2" est une chaîne de caractères composée du symbole deux qu'on ne peut pas soustraire, ou additionner, ... En revanche 2 est un nombre auquel on peut faire subir des opérations. Pour s'en convaincre, essayez l'instruction suivante

print("2"+"1")
print(2+1)

L'affichage à l'écran

Nous savons maintenant enregistrer la saisie clavier de l'utilisateur, nous allons voir comment afficher une réponse.

nom = "Parker"
prenom = "Peter"
print(prenom, nom) # affiche Peter Parker l'un en dessous de l'autre
print(prenom, nom, end="") # affiche Peter Parker
print(prenom + nom) # affiche PeterParker
print(prenom + "\n" + nom) # affiche Peter Parker l'un en dessous de l'autre

A vous de jouer : afficher votre nom puis votre prénom en utilisant chacune des différentes méthodes ci-dessus
