Les listes

📚 Compétence : savoir changer un élément dans une liste

------------------------------

Nous avons vu qu'une chaîne de caractères est un ensemble de caractères qu'on repère par un indice. On peut considérer une liste comme étant un ensemble d'éléments (textes ou nombres) qu'on repère par un indice.

Par exemple, ceci est une liste

maListe = ["lundi", "mardi", "mercredi", "jeudi", "vendredi"]

Définir une liste

Pour définir une liste vide, on a le choix entre deux possibilités

maListe = []
maListe = list()

Pour définir une liste contenant déjà des éléments, il suffit de les écrire en les séparant à l'aide d'une virgule. Attention, il faut penser à mettre des " pour chaque élément lorsqu'il s'agit d'une chaîne de caractères.

maListe = [3, 5, 2, 7, 8, 1]

A RETENIR ...

Dans une liste le 1er élément a pour indice 0 et non pas 1, c'est à dire qu'il se trouve à la position 0.

Les opérations élémentaires

Récupérer un élément

Pour récupérer l'élément d'indice i de maListe et le placer dans la variable jour :

jour = maListe[i]

Rajouter un élément en fin de liste

maListe.append("mon nouvel élément")

Changer un élément dans une liste

Pour remplacer l'élément d'indice i de maListe par "élément modifié" : 

maListe[i] = "élément modifié"

Exercice type : créer une liste contenant les noms "May", "Capucine", "Joaquim". Rajouter ensuite "Ethan", puis afficher l'élément d'indice 2 de cette liste.

listeNoms = ["May", "Capucine", "Joaquim"]
listeNoms.append("Ethan")
print(listeNoms[2])
