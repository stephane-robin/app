Afficher les éléments d'une liste

Inutile d'écrire quelque chose de compliqué pour simplement afficher les éléments d'une liste.

for elt in maListe:
    print(elt)

Manipuler les éléments d'une liste

Le nombre d'éléments de maListe est donné par l'instruction len(maListe).

Dans le code ci-dessous, on cherche à parcourir maListe et si on trouve l'élément "bonjour", alors on le modifie dans la liste en "hello".

for i in range(len(maListe)):
    if maListe[i] == "bonjour":
        maListe[i] = "hello"

Exercice type : créer un liste contenant les 10 premiers mois de l'année, puis rajouter Novembre et Décembre. Finalement, modifier chacun de ces mois en leur numéro correspondant.

Solution : 

maListe = ["Janvier", "Février", "Mars", "Avril", "Mai", "Juin", "Juillet", "Août", "Septembre", "Octobre"]
maListe.append("Novembre")
maListe.append("Décembre")
for i in range(len(maListe)):
    maListe[i] = i + 1

Tester l'appartenance à une liste

Considérons maListe = ["orange", "pamplemousse", "banane", "raisin"]

méthodesignification
"banane" in maListerenvoie True si le mot "banane" est un élément de la liste. Attention, cela ne teste pas si "banane" se trouve dans un élément de maListe
maListe.count("banane")renvoie le nombre de fois que "banane" est un élément de maListe
maListe.index("banane")renvoie le 1er indice de "banane" dans maListe

A vous de jouer : à partir de maListe définie ci-dessus, tester si "banane" appartient à cette liste, et si c'est le cas, afficher "il faut acheter du chocolat", sinon afficher "il faut acheter des bananes".
