La boucle for

📚 Compétence 8 : savoir utiliser une boucle for

------------------------------

Python permet de répéter une instruction un certain nombre de fois. Lorsqu'on connaît le nombre de répétitions, on utilise la boucle for. Lorsqu'on ne connaît pas le nombre de répétitions et que celui-ci dépend de l'évolution de la situation (par exemple si le nombre de vies d'un joueur devient nul), alors dans ce cas on utilise la boucle while.

Par exemple, pour répéter bonjour 5 fois, il suffit d'écrire

for i in range(5):
    print("bonjour")

Structure d'une boucle for avec range

Il est possible de personnaliser les répétitions en modifiant le pas ou le nombre de départ. En revanche, on ne peut augmenter le pas que d'un nombre entier. Si on souhaite utiliser un pas décimal, il faut alors utiliser l'instruction arange du module numpy. Testez les codes ci-dessous pour en comprendre le fonctionnement

from numpy import *

# répétition pour i allant de 0 à 6
for i in range(0, 7):
    print("bonjour")

# répétition allant de 2 à 8
for i in range(2, 9):
    print(i)

# répétition allant de 0 à 10 en augmentant de 2 à chaque tour
for i in range(0, 11, 2):
    print(i)

# répétition allant de 7 à 4 en descendant de -1 à chaque tour
for i in range(7, 3, -1):
    print(i)

# répétition allant de 0 à 1 en augmentant de 0.1 à chaque tour
for i in numpy.arange(0, 1.1, 0.1):
    print(i)

Parcourir une liste

La boucle for permet de parcourir une liste. On distingue deux structures différentes, l'une basée sur l'indice de chaque élément de la liste, l'autre basée directement sur chaque élément de la liste.

Les deux exemples ci-dessous fournissent le même résultat en utilisant des méthodes différentes. Il faut absolument connaître les deux méthodes, car elles sont plus ou moins pratiques en fonction de chaque situation.

maListe = ["lundi", "mardi", "mercredi", "jeudi", "vendredi"]

# méthode 1 basée sur l'indice i
for i in range(len(maListe)):
    print(maListe[i])

# méthode 2 basée directement sur l'élément lui-même
for elt in maListe:
    print(elt)

Exercice type : considérons la liste suivante
maListe = ["lundi", "mardi", "mercredi", "jeudi", "vendredi"]
1. Ajouter à chacun des éléments de la maListe la lettre s
2. Afficher chaque élément de maListe en y rajoutant la lettre s

Solution : 

# question 1 : maListe est modifiée
for i in range(len(maListe)):
    maListe[i] = maListe[i] + "s"

# qestion 2 : maListe est utilisée pour l'affichage mais n'est pas modifiée
for elt in maListe:
    print(elt + "s")
