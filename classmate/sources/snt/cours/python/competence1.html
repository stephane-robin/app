L'environnement Python

📚 Compétence 1 : découvrir l'environnement Python

------------------------------

Python est un langage de programmation interprété dont la philosophie repose sur la simplicité du code et l'implémentation de bonnes pratiques. Rien n'est fait pour empêcher le programmeur de mener une action, même si elle semble inappropriée. Python est particulièrement utilisé dans les environnements scientifiques et pour la manipulation de données.

Dans un langage interprété, le code source est traduit pas à pas par un logiciel appelé interpréteur, qui récupère des données d'entrée pour fournir des données de sortie. Le code source n'est pas transformé en code binaire comme pour un langage compilé. Le principal avantage est que le code source peut fonctionner sur tout ordinateur. Le principal inconvénient est qu'un langage interprété est plus lent qu'un langage compilé.

Environnement de développement

On peut exécuter directement un programme Python à partir du Terminal d'un ordinateur, mais il est plus pratique d'utiliser un éditeur de texte tel que Sublim Text qui colorie automatiquement le code en fonction du langage choisi et propose une auto-complétion du code. Il est encore plus efficace d'utiliser un Environnement de Développement Intégré tel que PyCharm qui propose également un debugger, une recherche passive des erreurs de frappe et le paramétrage des outils de versionning.

Ainsi, Python est installé sur votre ordinateur, mais est indépendant de l'éditeur ou de l'environnement que vous utiliser pour l'exécuter.

Vous pouvez également utiliser un interpréteur Python online sur Internet sans aucune instalation préalable. Il vous suffit par exemple de vous rendre sur le site OnlineGDB.

Découvrir l'environnement avec "Hello world"

Pour afficher "Hello world", exécutez dans votre environnement de développement :

print("Hello world")

Notez l'importance des "" dans le script ci-dessus. Essayez le même script sans les "". L'interpréteur Python pense dans ce cas que Hello est une variable dont il ne connaît pas la valeur. Les "" permettent simplement d'afficher du texte et non pas une valeur.
