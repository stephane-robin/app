Structure Mise en forme Propriétés Héritage

CSS

CSS signifie Cascading Style Sheets. Ce langage gère l'apparence des pages Web en arrangeant le contenu (couleur, marges, taille du texte). Comme il travaille le contenu d'une page Web, CSS utilise le code HTML pour fonctionner. Sans aucun code CSS, la page Web ressemble à du texte regroupé par bloc et tapé "au kilomètre".

Structure d'une page CSS

Il est possible de coder la mise en forme d'une page Web directement dans un fichier HTML sans recourir à un fichier CSS. Dans ce cas, on définit individuellement pour chaque balise la forme adéquate à l'aide de l'attribut style.

<p style="color:red"> mon texte </p>
HTML

Cette définition locale est coûteuse en temps et n'est conseillée qu'à titre exceptionnel. On peut également définir la mise en forme dans l'en-tête head d'un fichier HTML. Toutefois, la méthode préconisée est de définir la mise en forme dans un fichier séparé comportant l'extension .css. Il faut alors rajouter l'instruction suivante afin de lier notre page HTML à notre page CSS.

<link rel="stylesheet" href="style.css">
HTML

On constate que l'attribut href pointe vers l'adresse de la page CSS qui devra se situer dans le répertoire courant.

Les commentaires en CSS sont entourés des symboles /* et */. Ils peuvent être écrits sur une ou plusieurs lignes.

Appliquer une mise en forme

Pour appliquer un style particulier, il faut sélectionner le nom d'une balise sur laquelle le style va s'appliquer, puis les propriétés qu'on souhaite modifier ainsi que leurs valeurs. Chaque ligne de code se termine par ; et il existe un : entre une propriété et sa valeur. Si une modification concerne plusieurs balises, on peut sélectionner les noms de balises séparés de ,.

Attention, il faut indiquer le nom de la balise sans les symboles < et >.

h1, h2{
  color: blue;
}
CSS

Lorsqu'on veut appliquer une mise en forme différente pour une même balise en fonction des situations, on ajoute l'attribut class à cette balise dans le fichier HTML et on spécifie les propriétés de class dans le fichier CSS.

<p class="paragraphe1"> mon texte </p>
HTML
.paragraphe1{
  color: blue;
}
CSS

📌 Attention, dans ce cas il est important de ne pas oublier le . devant le nom de la classe de la balise.

Remarque , on peut également utiliser l'attribut id qui se construit de la même manière que l'attribut class. En revanche, dans le fichier CSS, il ne faut plus faire précéder le nom de la classe par un . mais l'identité par un #. L'attribut id a pour particularité de ne s'utiliser qu'une fois dans la page HTML.

Pour sélectionner une balise se trouvant à l'intérieur d'une autre balise, on ne met pas de , entre la balise père et la balise fils :

h2 strong{
  color: blue;
}
CSS

Les différentes propriétés

Les dimensions d'un bloc

On utilise les attributs width et height en pixels ou en pourcentage. On choisira préférentiellement le pourcentage afin de s'adapter au mieux aux différents formats d'écrans. Par défaut, un bloc prend 100% de la largeur disponible.

p{
  width:60%;
  height:10px;
}
CSS

La police

Pour qu'une police s'affiche correctement lors de l'ouverture de la page Web, il faut que le navigateur la connaisse. Il convient donc de choisir une police standard. Il est en fait conseillé de choisir plusieurs polices afin de s'assurer qu'une au moins de ces polices s'affiche. Le choix de police se fait naturellement selon l'ordre d'écriture de ces polices dans le fichier CSS.

font-family: Verdana, "Trebuchet MS", Arial;
CSS

A noter qu'on utilise des guillemets " dans le cas d'une police dont le nom contient plusieurs mots séparés par un espace.

La taille de police

On peut choisir de définir une taille absolue en pixels px ou relative en em. Il est conseillé d'utiliser une taille relative qui s'adapte mieux aux différents formats d'écrans. 1em correpond à une taille standard de texte ou de titre en fonction de la section dans laquelle on se trouve.

font-size: 1.2em;
CSS

La mise en forme standard

Pour mettre un texte en italique, en gras, souligner,... on utilise les propriétés suivantes :

font-style: italic;
font-style: normal;

font-weight: bold;
font-weight: normal;

text-decoration: blink;
text-decoration: underline;
text-decoration: line-through;
text-decoration: overline;
text-decoration: none;

font-variant: small-caps; /* petites capitales */
font-variant: normal;
CSS

L'alignement du texte dans un bloc

Pour aligner du texte, on utilise la propriété text-align.

text-align: left;
text-align: right;
text-align: center;
text-align: justify;
CSS

L'alignement avec l'attribut text-align ne fonctionne qu'avec les blocs et non pas les balises inline telles que <span>, <a>, <em>, <strong>,...

La rotation du texte

Pour tourner un texte d'un angle de 60 degrés, on utilise l'attribut transform.

#bloc{
  transform: rotate(60deg);
}
CSS

La couleur de texte

La couleur peut s'exprimer en notation hexadécimale ou en notation RGB pour Red Green Blue. Nous utiliserons principalement la notation hexadécimale, mais aussi la notation RGB lorsque la couleur est associée à une certaine opacité. Certains sites proposent de fournir l'équivalence entre couleurs et notations RGB ou hexadécimale. Par ailleurs, certaines extensions des navigateurs permettent de récupérer la notation RGB d'une couleur présente sur un site Web. On change la couleur d'un texte en utilisant le mot clé color :

color: rgb(153,102,102);
CSS

La couleur de fond

La couleur de fond se définit de la même manière que la couleur du texte, mais cette fois en utilisant le mot clé background-color:

background-color: rgb(255,255,253);
CSS

L'image de fond

Il est également possible d'appliquer une image de fond à la place d'une couleur de fond. Pour cela, il faut utiliser la propriété background-image, qui dispose de plusieurs options.

Attribut Effet obtenu 
background-image: url("nomImage.png");
 identité de l'image 
background-attachment: fixed;
 fixe le fond 
background-attachment: scroll;
 l'image de fond défile avec le texte 
background-repeat: no-repeat;
 le fond ne se répète pas, cela doit être combiné avec l'instruction background-position 
background-position: 40px 60px;
 indique où doit se trouver l'image de fond, comme l'origine des coordonnées se trouve en haut à gauche, l'image se trouve à 40 pixels de la gauche et à 60 pixels du haut 
background-position: top-right;
 dans ce cas l'image de fond se trouve en haut à droite

La transparence

Il est également possible de modifier la transparence d'une image ou d'une couleur à l'aide de la propriété opacity allant de 0 à 1. Par défaut, les éléments sont totalement opaques (c'est à dire non transparents) et opacity a pour valeur 1.

opacity: 0.6; /* la transparence est de 40% */
CSS

On peut également jouer sur la transparence en donnant une 4ème valeur comprise entre 0 et 1 à la propriété background-color. Attention, dans ce cas l'instruction rgb est remplacée par rgba.

background-color: rgba(255,255,253,0.6);
CSS

Les bordures

La propriété border permet de créer ou modifier une bordure et s'utilise avec des options indiquant la largeur, la couleur et le type. Par défaut, un élément ne comporte pas de bordure.

border: 2px red solid; /* trait plein rouge largeur 2 pixels */
border: 2px red dotted; /* trait pointillé */
border-radius: 10px; /* les bords sont arrondis de rayon 10 pixels */
box-shadow: 6px 6px 0px black; /* ombre de boîte horizontale, verticale,
fondue, couleur */
CSS

Les marges

Les marges intérieures au bloc sont définies en pixels par l'attribut padding alors que les marges extérieures (qui délimitent un espace autour du bloc) sont définies en pixels par l'attribut margin.

.monBloc{
  margin: 5px;
  padding: 10px;
}

.autreBloc{
  margin-top: 7px;
  margin-right: 5px;
  margin-bottom: 3px;
  margin-left: 5px;

  padding-top: 10px;
  padding-right: 15px;
  padding-bottom: 10px;
  padding-left: 15px;
}

.encoreUnBloc{
  margin: 7px 0 6px 3px;
  padding: 10px 15px 15px 10px;
}
CSS

Dans le dernier cas ci-dessus, l'ordre de lecture est vraiment important. top, right, bottom, left.

Centrer un bloc

Pour centrer horizontalement un bloc, il faut utiliser l'instruction margin: auto; conjointement avec l'attribut obligatoire width pour définir une largeur de bloc avant de le centrer. Il n'est pas possible de centrer verticalement un bloc avec cette méthode. Il convient alors d'utiliser Flexbox pour centrer verticalement.

p{
  width: 300px;
  margin: auto;
}
CSS

Lorsque le texte dépasse la taille du bloc

Lorsqu'un bloc a des dimensions précises et que son contenu dépasse les bords, on utilise l'instruction overflow: auto pour gérer le dépassement automatiquement et créer une barre de défilement si besoin.

De façon similaire, lorsqu'un mot très long dépasse les bords, on utilise l'instruction word-wrap: break-word pour couper le mot sans en perdre la signification. C'est notamment le cas d'adresses Internet très longues.

Les listes

On peut définir le type de liste comme un attribut des balises <ul> et <ol>.

ul{
  list-style-type: disc;
  list-style-type: circle;
  list-style-type: square;
  list-style-type: none;
  list-style-image: url('nomImage.png'); /* pour une puce personnalisée dont
  l'image devra être dans le dossier courant */
}

ol{
  list-style-type: decimal;
  list-style-type: lower-roman;
  list-style-type: upper-roman;
  list-style-type: lower-alpha;
  list-style-type: upper-alpha;
}
CSS

Les flottants

Il est possible d'insérer un élément flottant devant un texte, qui pourra être un bloc ou un image par exemple. Il faut pour cela définir des marges et largeurs compatibles afin que le texte ne déborde pas.

.blocTexte{
  margin-left: 120px;
}

.blocFlottant{
  float: left;
  width: 100px;
}
CSS

L'instruction clear: both oblige la suite du texte à se placer sous le bloc flottant.

Les positionnements absolus, fixes et relatifs

Le positionnement absolute permet de placer un élément sur une page. On utilise comme unité de mesure des pixels ou des pourcentages. Notons que l'origine du repère se trouve toujours en haut à gauche du bloc qui contient le bloc à placer. Initialement, l'origine du repère se trouve donc en haut à gauche de l'écran.

.bloc{
  position: absolute;
  top: 10%;
  right: 30%;
}
CSS

En cas de chevauchement de blocs, l'attribut z-index permet de déterminer quel bloc doit apparaître au-dessus des autres. Le bloc ayant la valeur de z-index la plus élevée sera placé au-dessus des autres.

.bloc1{
  position: absolute;
  top: 10%;
  right: 30%;
  z-index: 1;
}

.bloc2{
  position: absolute;
  top: 10%;
  right: 35%;
  z-index: 2;
}
CSS

Le positionnement fixed permet de placer un élément sur une page et de s'assurer qu'il reste toujours visible même si on se déplace sur la page. C'est utile notamment si l'on souhaite qu'un bandeau reste en haut d'une page lorsqu'on se déplace avec la souris.

Le positionnement relative permet de décaler l'élément par rapport à la position d'origine. A éviter par souci de simplicité.

Modifier l'aspect d'un lien lorsque la souris le survole

Il est possible de modifier toutes les propriétés d'un lien lorsqu'on le survole en utilisant l'instruction a:hover.

a:hover{
  color: red;
  font-weight: bold;
}
CSS

📌 Attention, dans ce cas il ne faut surtout pas mettre d'espace entre a et hover sous peine de dysfonctionnement car hover n'est pas la valeur d'un attribut.

Les tableaux

Pour définir les bordures d'un tableau, on utilise les instructions suivantes :

table{
  border-collapse: collapse; /* pour que les bordures des cellules se collent */
}

th, td{
    border: 1px solid black;
}
CSS

Il n'est pas nécessaire de créer des bordures pour la balise <tr> car elles ont déjà été faites avec les bordures <th> et <td>.

Pour changer la position du titre du tableau : 

table{
  caption-side: top;
  caption-side: bottom;
}
CSS

Notion d'héritage

La notion d'héritage est issue de la programmation orientée objet. En CSS, cela signifie que si on applique une propriété à une balise, toutes les balises qui se trouvent à l'intérieur appliqueront cette même propriété. On dit qu'elles héritent de cette propriété. Par exemple si on définit la couleur du texte à l'intérieur d'un paragraphe comme étant bleue, tous les mots mis en valeur par la balise em à l'intérieur d'un paragraphe seront bleus, sauf si on redéfinit spécifiquement la couleur de la balise em.

Le code suivant est interprété de la façon suivante par le navigateur, où le mot "Attention" se trouvant à l'intérieur de balises <em> est colorié en bleu.

<p style="color:rgb(51,51,255)"><em>Attention</em>, je tape mon texte ici.</p>
HTML

associé à

p{
  font-family: Verdana, "Trebuchet MS", Arial;
  text-align: justify;
  color: rgb(51,51,255);
}
CSS

qui produit l'affichage suivant

Attention, je tape mon texte ici.

En revanche, le code suivant est interprété différemment, et la définition spécifique de la couleur de la balise <em>, outrepasse l'héritage issu de la balise <p>. 

<p style="color:rgb(51,51,255)"><em style="color:rgb(255,51,102)">Attention</em>, je tape mon texte ici.</p>

associé à

em{
  color: rgb(255,51,102);
}
CSS

qui produit l'affichage suivant

Attention, je tape mon texte ici.
